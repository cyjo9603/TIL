# 자료형

### 동적 타이핑

- 자바스크립트는 약타입 언어, 혹은 동적 타입 언어이다.
- 그러므로 변수의 타입을 미리 선언할 필요가 없이 프로그램이 컴파일 되는 과정에서 자동으로 파악이 된다.
- 즉, 같은 변수에 여러 타입의 값을 넣을 수 있다.

# 원시 타입 ( Primitive values )

## 1. Boolean

- Boolean은 논리적인 요소를 나타내고, `true`와 `false`의 두 가지 값을 가질 수 있다.

## 2. Null

- Null타입은 딱 한 가지 값, `null`을 가질 수 있다.
- 모든 객체가 `null`값에서 파생 되므로 `typeof` 검사기는 `object` 타입을 반환한다.

```jsx
typeof null === 'object'; // true
```

## 3. Undefined

- 값을 할당하지 않은 변수는 `undefined`값을 가진다.

## 4. Number

- Number타입은 정수 및 부동소수점 숫자(floating point number)를 나타낸다.
- 숫자형과 관련된 연사는 다양한데, 곱셈`*`, 나눗셈`/`, 덧셈`+`, 뺄셈`-` 등이 대표적이다.
- 숫자형엔 일반적인 숫자 외에 `Infinity`, `-Infinity`, `NaN`같은 특수 숫자 값이 포함된다.

- `Infinity`는 어떤 숫자보다 큰 특수 값, 무한대(∞)를 나타낸다.

    어느 숫자든 `0`으로 나누면 `Infinity`값을 얻을 수 있다.

    ```jsx
    1 / 0 // Infinity
    Infinity // Infinity

    1 / -0 // -Infinity
    -Infinity // -Infinity
    ```

- `NaN`은 계산 중에 에러가 발생했다는 것을 나타내주는 값으로 부정확하거나 정의되지 않은 수학 연산을 사용하면 계산 중에 에러가 발생하는 데, 이때 `NaN`이 반환된다.
- `NaN`은 여간해선 바뀌지 않고 `NaN`에 어떤 추가 연산을 해도 결국 `NaN`이 반환된다.

    ```jsx
    'text' / 2 // NaN
    ```

자바스크립트에서 행해지는 수학 연산은 '안전'하다고 볼 수 있다. 0으로 나눈다거나 숫자가 아닌 문자열을 숫자로 취급하는 등의 이례적인 연산이 자바스크립트에선 가능하다. 말이 안되는 수학 연산을 하더라도 스크립트는 치명적인 에러를 내뿜으며 죽지 않고 `NaN`을 반환하며 연산을 종료시킨다.

### BigInt

- 내부 표현 방식 때문에 자바스크립트에선 `(2^53-1)(9007199254740991)` 보다 큰 값 혹은 `-(-2^53-1)` 보다 작은 정수는 Number타입을 사용해 나타낼 수 없다.
- 사실 대부분의 상황에서 이런 제약사항은 문제가 되지 않지만 암호 관련 작업같이 아주 큰 숫자가 필요한 상황이거나 아주 높은 정밀도로 작업을 해야 할 때는 이런 큰 숫자가 필요하다.
- `BigInt`형은 표준으로 채택된 지 얼마 안된 자료형으로, 길이에 상관없이 정수를 나타낼 수 있다.
- `BigInt`형 값은 정수 리터럴 끝에 `n`을 붙이면 만들 수 있다.

```jsx
const bigInt = 123456789012345678901234567890n;
```

BigInt는 Firefox와 Chrome에서만 지원되며 Safari, IE, Edge 등에선 지원되지 않는다.

## 5. String

- 자바스크립트에선 문자열을 따옴표로 묶는다.
- 따옴표는 세 종류가 있다.
    1. 큰 따옴표: `"string"`
    2. 작은 따옴표: `'string'`
    3. 역 따옴표(백틱, backtick): ``string``
- 큰 따옴표와 작은 따옴표는 기본적인 형식으로, 자바스크립트에서는 이 둘에 차이를 두지 않는다.
- 역 따옴표로 변수나 표현식을 `${...}`으로 감싸면 원하는 변수나 표현식을 문자열 중간에 넣을 수 있다.

## 6. Symbol

- `Symbol`타입은 ES6에서 추가되었으며 유일하고 변경 불가능한 원시 값이다.
- 객체 속성의 key값으로도 사용될 수 있으며 몇몇 프로그래밍 언어에서는 atom이라고 부른다.

# 참조 타입 ( Reference type )

- 원시 타입이 아닌 모든 값은 참조 타입의 값이다.
- 객체, 배열, 함수 등의 타입이며, 메모리에 값을 주소로 저장하고, 출력시 메모리 주소와 일치하는 값을 출력한다.

### Object

- Object타입은 특수한 자료형으로 데이터 컬렉션이나 복잡한 개체를 표현할 수 있다.
- 이런 특징 때문에 자바스크립트에서 객체는 좀더 특별한 취급을 받는다.
